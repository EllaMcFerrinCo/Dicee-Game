# Dicee Game

A two player player game where whoever rolls the largest number wins.

![Image](/images/Screenshot1.png)


## Structure

This game is made up of:
- html page
- javascript
- css

## Implementation

Everytime you refresh the page:
1. two random numbers are generated between 1-6
2. the images are then assigned based on these random numbers
3. the greater number is displayed as the winner

## How to Run

1. Clone or download this repository
2. Open index.html file in your browser
3. refresh page to roll the dice
